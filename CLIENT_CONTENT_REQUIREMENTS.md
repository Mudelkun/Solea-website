## Notes for Content Submission

1. **Text Format**: Please provide text in a Word document or Google Doc
2. **Images**: 
   - Minimum resolution: 1200px width for large images, 800px for products
   - Formats: JPG, PNG, or WebP
   - Please name files descriptively (e.g., "product-shampoo-front.jpg")
3. **Organization**: Group content by section for easier processing
4. **Timeline**: Please indicate priority items if everything cannot be provided at once
